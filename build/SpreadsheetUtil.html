static array loadFileIntoArray(filename $filename, hasHeaders $hasHeaders = false)

Load file into an associative array that contains the file contents.

The parent array has two keys: "headers" and "cells". The "headers" key is still present even if $hasHeaders is false.

Each key of the "headers" array is the column (e.g. "A", "B", "C")

Each key of the "cells" array is the row number (e.g. 0, 1, 2). The rows are a zero-index array. Each key of the array that represents a row is the column (e.g. "A", "B", "C").

To access cell "A1", where $sheetData is the result of this function:

$sheetData['cells'][0]['A']

Parameters

filename $filename string path to the spreadsheet file
hasHeaders $hasHeaders bool If the first row of the file is column names (default = false)

Return Value

array An associative array
